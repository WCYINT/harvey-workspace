# Thesis Retro - 写作过程复盘

## 复盘维度

| 维度 | 问题 |
|------|------|
| **过程** | 写作流程是否高效？ |
| **质量** | 哪些部分质量最高？ |
| **挑战** | 遇到的最大困难是什么？ |
| **改进** | 下次如何做得更好？ |

## 复盘框架

### 1. 过程回顾
- 用了多长时间？
- 各章节时间分配？
- 最高效的部分？

### 2. 质量评估
- 哪些章节质量最高？
- 哪些需要重写？
- 数据支撑是否充分？

### 3. 挑战识别
- 最大困难是什么？
- 花了最多时间的部分？
- 哪些知识欠缺？

### 4. 改进计划
- 下次如何优化？
- 需要加强哪些技能？
- 时间如何重新分配？

## 使用方法

输入 /retro，我来帮你复盘论文写作过程。
