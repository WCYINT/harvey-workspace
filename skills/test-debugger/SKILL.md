# debugger
Professional debugger tool with full CLI support. Use when you need debugger functionality from the terminal.
## Usage
```
debugger analyze <input>
debugger export <format>
debugger view <file>
```
## Commands
- `analyze` — Analyze and process input data
- `export` — Export results in various formats  
- `view` — View and inspect content
## Requirements
- bash
- python3
## When to Use
Use when working with debugger tasks in a terminal environment.
