# Evolver Skill

Enables OpenClaw agents to evolve and improve their capabilities over time.

## Capabilities

1. **Capability Growth** - Analyze and improve agent capabilities
2. **Skill Composition** - Combine multiple skills for complex tasks
3. **Adaptive Behavior** - Modify behavior based on feedback
4. **Performance Optimization** - Optimize agent performance

## Usage

When user mentions evolution, growth, or capability improvement:
- Activate evolver system
- Analyze current capabilities
- Suggest improvements
- Implement adaptive changes

## Integration

This skill integrates with:
- `self-improving-agent` - Learning and feedback
- `brainstorming` - Requirement exploration
- `openclaw-agent-optimize` - Performance tuning
