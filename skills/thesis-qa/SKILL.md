# Thesis QA - 论文质量检查

## 三层审查模式

| 模式 | 覆盖范围 | 适用场景 |
|------|---------|---------|
| **Quick** | 关键问题：格式、逻辑断裂、数据异常 | 时间紧张 |
| **Standard** | Quick + 中等问题：引用、图表、论证 | 常规审查 |
| **Exhaustive** | Standard + 全面检查：语言、风格、完整性 | 定稿前 |

## 检查清单

### 1. 格式规范
- [ ] 字体、字号、行距统一
- [ ] 页眉页脚正确
- [ ] 中英文目录完整
- [ ] 章节编号一致

### 2. 逻辑完整性
- [ ] 假设→论证→结论 链条完整
- [ ] 章节间逻辑衔接
- [ ] 因果关系清晰
- [ ] 无逻辑跳跃

### 3. 数据一致性
- [ ] 摘要数据与正文一致
- [ ] 图表数据与描述一致
- [ ] 前后文数据无矛盾

### 4. 引用规范
- [ ] 参考文献格式统一 (GB/T 7714)
- [ ] 引用与参考文献对应
- [ ] 引用位置正确

### 5. 图表完整性
- [ ] 图表编号连续
- [ ] 图题/表题完整
- [ ] 图表引用正确

### 6. 语言表达
- [ ] 学术用语规范
- [ ] 无口语化表达
- [ ] 术语统一

## 使用方法

1. 告诉我想用哪种模式 (Quick/Standard/Exhaustive)
2. 提供论文文件或具体问题
3. 我进行全面检查并报告问题
