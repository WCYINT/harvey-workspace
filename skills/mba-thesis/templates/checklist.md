# MBA论文修改检查清单

## 结构检查
- [ ] 封面信息完整准确
- [ ] 摘要中英文齐全
- [ ] 目录自动生成且页码正确
- [ ] 章节标题层级清晰
- [ ] 逻辑结构连贯

## 内容检查
- [ ] 研究问题明确具体
- [ ] 文献综述全面深入
- [ ] 理论框架完整合理
- [ ] 研究方法科学适当
- [ ] 数据分析严谨可靠
- [ ] 结论与建议相关

## 格式检查
- [ ] 字体、字号统一（宋体/Time New Roman）
- [ ] 行距一致（1.5倍或2倍）
- [ ] 页边距符合要求（通常2.5-3cm）
- [ ] 页码位置正确
- [ ] 图表编号规范（图1-1，表1-1）
- [ ] 图表标题清晰

## 引用检查
- [ ] 正文引用与参考文献一致
- [ ] 参考文献格式统一（APA/MLA/Chicago）
- [ ] 参考文献数量充足（通常50-100篇）
- [ ] 中英文文献比例适当

## 语言检查
- [ ] 语法正确
- [ ] 拼写无误
- [ ] 标点符号规范
- [ ] 学术语言正式
- [ ] 避免口语化表达
- [ ] 逻辑连接词恰当

## 学术规范
- [ ] 无抄袭嫌疑
- [ ] 数据真实可靠
- [ ] 研究伦理合规
- [ ] 致谢内容恰当

## 最终检查
- [ ] 全文通读一遍
- [ ] 格式转换测试（PDF）
- [ ] 打印预览检查
- [ ] 符合学校具体要求
