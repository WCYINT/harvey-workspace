After subscribing:

```bash
export DEVELOPER_KIT_KEY="your-subscription-key"

```

## Usage Examples

**Start a new project:**
```
"我想做一个 React Todo 应用"
→ 自动调用 react-expert + typescript + testing-patterns
→ 输出：组件架构 + 类型定义 + 测试模板
```

**Review a PR:**
```
"帮我审查这个 PR"
→ 自动调用 github-repo-analyzer + typescript + react-expert
→ 输出：代码质量分析 + 类型安全检查 + 组件设计建议
```
