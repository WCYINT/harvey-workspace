# 📅 AI日报生成工具 - OpenClaw Skill
自动生成AI行业日报的OpenClaw技能，覆盖全球AI动态、技术突破、产品发布、行业应用、OPC政策动态等核心信息，支持飞书消息格式，可直接定时推送到群聊或个人。

## ✨ 核心功能
- 🕒 **每日自动更新**：严格筛选过去24小时内的权威AI资讯
- 📊 **八大内容板块**：巨头动态/技术突破/产品发布/一人公司政策/政务应用/行业机遇/垂直应用/行业资讯
- 🏢 **OPC政策专项**：优先收录各地OPC针对AI个体创业者的扶持政策动态
- ✅ **信息验证**：多渠道交叉验证，确保信息准确性和权威性
- 📝 **结构化输出**：自动生成包含今日要点、分板块详情、数据来源的完整日报
- 💬 **飞书适配**：直接输出飞书富文本格式，可直接推送无需二次排版
- ⚙️ **自定义支持**：可聚焦特定领域生成专属日报

## 🚀 快速使用

```

```

## 🛠️ 安装使用
### 环境要求
- OpenClaw >= 1.0.0
- 已配置网络访问权限
- 飞书集成（可选，用于推送消息）

### 安装方式
1. 下载技能包
2. 解压到OpenClaw技能目录：`/workspace/projects/workspace/skills/`
3. 重启OpenClaw服务后即可使用
