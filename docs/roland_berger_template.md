# Roland Berger 咨询公司工作方式参考

## 公司特点
- 全球顶级战略咨询公司
- 专注于战略、创新、转型
- 数据驱动的分析方式
- 注重实际落地执行

## 工作方法论

### 1. 分析框架
- 使用结构化分析工具（如波特五力、SWOT、PEST等）
- 数据驱动的决策支持
- 跨行业最佳实践对比

### 2. 项目流程
1. **问题定义** - 明确核心挑战
2. **深度调研** - 行业、竞争、市场分析
3. **方案设计** - 战略/运营/数字化方案
4. **实施支持** - 落地辅导与持续优化

### 3. 交付物格式标准

| 交付物 | 格式要求 |
|--------|----------|
| **Executive Summary** | 1页摘要，核心结论+行动建议 |
| **分析报告** | 详细数据+图表+洞察 |
| **方案PPT** | 结论先行，逻辑清晰，视觉化 |
| **实施路线图** | 时间线+里程碑+责任人 |

## Havery 交付物标准（参照罗兰贝格）

### 📊 报告格式

```
# [标题]

## 执行摘要 (Executive Summary)
- 核心发现：...
- 关键建议：...
- 预期效果：...

## 分析详情
### 1. [主题1]
- 数据/事实
- 分析
- 洞察

### 2. [主题2]
...

## 行动建议
| 优先级 | 行动项 | 负责人 | 时间线 |
|--------|--------|--------|--------|
| P1 | ... | ... | ... |

## 附件
- 详细数据
- 方法论说明
```

### 📋 表格格式
使用清晰的表格呈现数据和对比

### 🎯 视觉元素
- 使用图表、图形增强理解
- 关键数字突出显示
- 逻辑层次分明

### 📝 语言风格
- 结论先行
- 数据支撑
- 简洁专业
- 行动导向

## 参照执行

每次交付物应包含：
1. **1页执行摘要** - 核心结论
2. **结构化分析** - 数据+洞察
3. **明确建议** - 可执行步骤
4. **时间线** - 实施计划
