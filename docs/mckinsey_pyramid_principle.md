# 麦肯锡金字塔原理 (The Pyramid Principle)

## 核心概念

金字塔原理是一种结构化思考和表达的方法论，由芭芭拉·明托(Barbara Minto)创立，被麦肯锡等顶级咨询公司广泛采用。

## 基本结构

```
     核心结论 (结论先行)
          │
    ┌─────┴─────┐
    │           │
  论据1       论据2
  (MECE)      (MECE)
 ┌──┴──┐    ┌──┴──┐
支撑1 支撑2 支撑3 支撑4
```

## 核心原则

### 1. 结论先行 (Summarize First)
- 最重要结论放在最前面
- 开门见山给出答案
- 再提供支撑论据

### 2. MECE 原则 (Mutually Exclusive, Collectively Exhaustive)
- 相互独立，完全穷尽
- 不重叠，不遗漏
- 分类清晰完整

### 3. 逻辑归纳
- 上层结论 = 下层论据的归纳
- 论据之间逻辑递进
- 同一层级同一范畴

## 表达结构

### 自上而下法 (演绎法)
1. 核心结论
2. 关键论据 (2-7个)
3. 支撑数据/案例

### 自下而上法 (归纳法)
1. 收集信息
2. 提炼观点
3. 归纳总结
4. 验证结论

## 应用场景

| 场景 | 结构 |
|------|------|
| 商业报告 | 背景→问题→核心建议→详细分析 |
| 演讲汇报 | 结论→要点→支撑 |
| 日常沟通 | 结论→原因→建议 |
| 问题分析 | 现象→原因→对策 |

## Havery 应用指南

### 分析问题时的思考流程

```
1. 是什么？(What)
   - 核心问题/现象
   
2. 为什么？(Why)
   - 根本原因分析
   - MECE分类
   
3. 怎么办？(How)
   - 解决方案
   - 优先级排序
   
4. 价值？(So What)
   - 预期效果
   - 资源需求
```
