## 一、D OE核心概念

### 1. 什么是DOE？
DOE（Design of Experiments）是通过**系统性地操作自变量(X)**，经济有效地得到关于过程响应的正确结论的统计工具。

### 2. 核心元素
| 概念 | 说明 |
|------|------|
| **因子 (Factors)** | 自变量 x1, x2, ..., xn，可人为控制 |
| **水平 (Levels)** | 因子的人为设定状态，通常每因子2水平（-1/+1） |
| **响应变量 (Response y)** | 因变量，测量因子变化的效应 |
| **效应 (Effect)** | 响应变量的差异 |
| **噪声因子 (z)** | 不可控因素，正常过程难以控制 |

### 3. DOE四大目标
1. **筛选**：确定对y影响最大的x
2. **瞄准**：确定x值使y接近目标额定值
3. **稳健**：确定x值使y的变异性最小
4. **免疫**：确定x值使不可控变量z的效应最小

---

## 二、三种试验策略对比

| 策略 | 优点 | 缺点 |
|------|------|------|
| 最佳猜测法 (Best-guess) | 简单 | 盲目，效率低 |
| 一次一因子 (OFAT) | 符合直觉 | **不考虑交互作用**，效率低 |
| **统计试验设计 (DOE)** | 同时探查多因子+交互 | 需要培训 |

## 三、全因子实验 (Full Factorial)

### 试验次数计算
- **2因子2水平**：2² = **4** 次试验
- **3因子2水平**：2³ = **8** 次试验
- **n因子2水平**：**2ⁿ** 次试验

### 设计矩阵示例（3因子，2水平）
| Trial | Factor A | Factor B | Factor C | Rep1 | Rep2 |
|-------|----------|----------|----------|------|------|
| 1 | -1 | -1 | -1 | y1 | y2 |
| 2 | +1 | -1 | -1 | y1 | y2 |
| ... | ... | ... | ... | ... | ... |
| 8 | +1 | +1 | +1 | y1 | y2 |

**关键：每种组合重复2次（Rep）以估计实验误差**

## 四、D OE七步操作流程

1. **识别问题与状态**
2. **筛选响应变量 (y)**
3. **选择因子、水平与范围**
4. **选择试验设计方案**
5. **实施试验**
6. **统计分析** (Minitab)
7. **得出结论与建议**

### 核心输出解读

#### 1. Pareto图
- 超过红线的效应 = **统计显著** (P < 0.05)
- 主效应 + 交互效应都要检查

#### 2. 效应表
```
Term                  Effect    Coef    T      P
Constant              57.000   0.7706  73.97  0.000
Temperature           39.500  19.750   0.7706  25.63  0.000
Pressure               6.000   3.000   0.7706   3.89  0.018
Temperature*Pressure  19.500   9.750   0.7706  12.65  0.000
```

- **P < 0.05 = 显著**
- R-Sq = 99.52% → 模型解释力极强

#### 4. 图形分析

## 六、稳健设计 (Taguchi方法)

### 核心思想
用最低成本元件组装高质量、高可靠性产品。

### 三类质量损失函数
| 类型 | 特征 | 例子 |
|------|------|------|
| **望目特性** | 有目标值，波动越小越好 | 零件尺寸 |
| **望小特性** | 希望值=0，越小越好 | 缺陷数 |
| **望大特性** | 越大越好 | 强度、寿命 |

### 三次设计
1. **系统设计**：概念/功能设计（专业领域知识）
2. **参数设计**：最优参数组合（核心环节，优化性能+成本）
3. **容差设计**：确定主次因素，控制成本

### 可控因子 vs 噪声因子
- 选择可控因子使响应变量对噪声**不敏感**
- 利用可控因子与噪声因子的**交互关系**实现稳健

## 七、相关统计方法

### t检验（两样本）
- 用途：判断两个方案/处理是否有**显著差异**
- 原假设 H0: μ₁ = μ₂
- 结论：P < 0.05 → 拒绝H0 → 差异显著

### 方差分析 (ANOVA)
- 用途：检验**k(k>2)个处理**均值差异
- H0: μ₁ = μ₂ = ... = μk
- F = MS(Factor) / MS(Error)，F大 → 处理间差异显著

### 回归分析
- 用途：建立 **Y = a + bX** 数量关系
- R-Sq：模型解释Y变化的百分比
- 残差诊断：检测异常值和模型假设

## 八、MBA论文应用建议

DOE在管理研究中的应用场景：
1. **生产管理**：工艺参数优化（温度、压力、时间）
2. **运营管理**：流程改善前后的效果验证
3. **质量管理**：供应商评价、原料选择
4. **人力资源**：培训方法、激励措施效果评估

**推荐章节结构：**
- 实证研究方法 → 实验设计 → DOE + ANOVA
- 保障措施章节 → 用DOE结果证明改善方案有效性
