# 学术写作技巧

## 语言风格
1. **正式性**
   - 避免口语化表达
   - 使用完整句子
   - 避免缩写（如"can't"应写为"cannot"）

2. **客观性**
   - 使用第三人称
   - 避免主观判断
   - 基于证据得出结论

3. **精确性**
   - 使用专业术语
   - 定义关键概念
   - 避免模糊表述

## 段落结构
1. **主题句**：每段开头明确本段主旨
2. **支持句**：提供证据、数据、例子
3. **结论句**：总结本段，连接下一段

## 常见问题与改进

### 1. 冗长表达
- **避免**：由于这个原因，我们可以得出结论...
- **改进**：因此，...

### 2. 被动语态过度使用
- **避免**：数据被收集...
- **改进**：研究者收集了数据...

### 3. 模糊表述
- **避免**：很多人认为...
- **改进**：Smith (2020)的研究表明...

## 连接词使用

### 表示因果关系
- 因此，所以，因而，由此
- because, therefore, thus, hence

### 表示对比关系
- 然而，但是，相反
- however, but, conversely

### 表示递进关系
- 此外，而且，更重要的是
- furthermore, moreover, additionally

### 表示举例说明
- 例如，比如，譬如
- for example, for instance, such as

## 学术词汇替换

| 口语化表达 | 学术化表达 |
|------------|------------|
| 很多 | 大量，众多，相当数量 |
| 很好 | 显著，有效，优越 |
| 发现 | 揭示，表明，证实 |
| 做研究 | 开展研究，实施调查 |
| 看数据 | 分析数据，检验结果 |

## 引用技巧
1. **直接引用**：使用引号，注明页码
2. **间接引用**：转述观点，注明作者年份
3. **综合引用**：整合多个来源观点

## 避免学术不端
1. **正确引用**：所有非原创内容必须引用
2. **避免抄袭**：即使改写也要注明来源
3. **数据真实**：不伪造、篡改数据

## 修改建议
1. **初稿**：关注内容和结构
2. **二稿**：优化语言和逻辑
3. **终稿**：检查格式和细节
4. **最终检查**：通读全文，确保连贯
